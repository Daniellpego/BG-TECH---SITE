# Organização do repositório

Este repositório foi limpo e agora possui apenas duas pastas principais:

- painel-crm/: Backend, frontend, documentação, infraestrutura, scripts e configs do CRM
- site-principal/: Site institucional, arquivos estáticos, HTML, CSS, imagens e scripts

Todos os scripts, configs e arquivos importantes foram migrados para painel-crm/scripts.

Pastas técnicas (.git, .github, node_modules) permanecem na raiz.

Para adicionar novos arquivos, mantenha-os dentro de uma dessas duas pastas principais.
